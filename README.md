# مسودة (Meswada)

مسوَّدة هو نظام إدارة محتوى محلي (Mini-CMS) مبني باستخدام بايثون (Python) وإطار عمل فلاسك (Flask)، صُمم خصيصاً للكُتّاب وصناع المحتوى التقني. يوفر لك بيئة كتابة هادئة ومنظمة تعمل بالكامل على جهازك الشخصي، مما يضمن لك السرعة والخصوصية التامة. يعتمد البرنامج في واجهته على محرر Editor.js الحديث، ليقدم لك تجربة كتابة سلسة ومتقدمة تحول أفكارك تلقائياً إلى كود HTML نظيف وجاهز للنشر المباشر.

## مميزات البرنامج

* **محرر نصوص متطور (Editor.js):** تجربة كتابة حديثة تعتمد على نظام "الكتل" (Blocks). يدعم إضافة العناوين، القوائم المتداخلة، الجداول، الأكواد البرمجية، والاقتباسات بكل سهولة واحترافية.

* **تنظيم هرمي ذكي:** رتب محتواك بسهولة من خلال إنشاء "مواقع" مستقلة، وتفريعها إلى "أقسام" فرعية، مما يجعل إدارة مئات المقالات أمراً في غاية البساطة.

* **تصدير مباشر إلى HTML:** بضغطة زر واحدة، يتم تحويل مسودتك بالكامل إلى كود HTML دقيق ونظيف، جاهز للنسخ واللصق في موقعك الحي.

* **مكتبة وسائط متكاملة:** *  ارفع الصور وأدرجها في مقالاتك مباشرة. يقوم النظام تلقائياً بتنظيم الصور في مجلدات مخصصة لكل موقع وقسم، مع إمكانية إدارتها وحذفها من لوحة تحكم المكتبة.

* **أرشيف وحفظ موثوق:** يعتمد البرنامج على قاعدة بيانات محلية (SQLite) خفيفة وسريعة تضمن حفظ مقالاتك بشكل آمن، مع توفر صفحة أرشيف للبحث والرجوع لأي مقال سابق وتعديله.

* **العمل بدون إنترنت:**  الأداة تعمل بنسبة 100% على بيئتك المحلية (Localhost)، مما يمنحك حرية الكتابة في أي وقت وأي مكان، مع ضمان بقاء بياناتك على جهازك فقط.

* **واجهة احترافية خالصة:**  تصميم بسيط وعملي يركز على المحتوى فقط، بدون أي تشتيت بصري أو إضافات معقدة.

## متطلبات التشغيل

* نظام تشغيل Linux (التعليمات أدناه مخصصة لتوزيعة Debian أو Ubuntu)
* Python 3
* بيئة بايثون الوهمية (venv)

## طريقة التشغيل على لينكس ديبيان (Debian Linux)

**1. تحديث النظام وتثبيت المتطلبات الأساسية**
تأكد من تثبيت لغة Python، وأداة تثبيت الحزم `pip`، وأداة إنشاء البيئات الوهمية `venv`:
```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv

cd /path/to/Meswada

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python run.py
