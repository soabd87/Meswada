# مسودة (Meswada)

نظام إدارة محتوى مصغر (Mini-CMS) محلي، مصمم خصيصاً لتسهيل سير عمل الكتابة التقنية. يعتمد على إطار عمل Flask في الواجهة الخلفية ومحرر Editor.js في الواجهة الأمامية، مع التركيز على سرعة الإنتاجية، التنظيم الهرمي للمحتوى، والتصميم البسيط والعملي.

## مميزات البرنامج

* **محرر نصوص متقدم:** تكامل كامل مع `Editor.js` يدعم العناوين، الأكواد البرمجية، الصور، الجداول، الاقتباسات، التضمين (YouTube, Twitter, Vimeo)، والقوائم المتداخلة بكفاءة.
* **هيكلة تنظيمية مرنة:** إدارة المحتوى عبر تسلسل هرمي دقيق: المواقع (Sites) -> الأقسام (Categories) -> المقالات (Posts)، مع إمكانية تعديلها أو نقل البيانات بينها بأمان دون فقدانها (Safe Delete).
* **تصدير HTML مخصص للمطورين:** تصدير المقالات بصيغة HTML جاهزة، حيث يتم دمج الصور بمساراتها المحلية المطلقة (Absolute Local Paths). يتيح ذلك نسخ المسار ولصقه مباشرة في نافذة اختيار الملفات (OS file-picker) عند الرفع الفعلي.
* **إدارة وسائط ذكية وديناميكية:** * تنظيم تلقائي للصور المرفوعة في مجلدات متداخلة بناءً على اسم الموقع والقسم `Site/Category`.
  * مكتبة وسائط شاملة تتيح استعراض الصور المرفوعة ونسخ مسارها المحلي الكامل بضغطة زر للوصول السريع.
* **إعدادات عامة مرنة:** لوحة تحكم لإدارة الإعدادات (مثل `local_export_path`) وتطبيقها برمجياً دون الحاجة لتعديل الأكواد الثابتة (Hardcoded).
* **عمليات إدارة سريعة:**
  * دعم عمليات الحذف المجمع (Bulk Delete) للمقالات لتوفير الوقت.
  * اختصارات لوحة المفاتيح: `Ctrl+S` (للحفظ مع مؤشر مرئي)، و `Ctrl+E` (للتصدير والنسخ التلقائي للنص البرمجي).
* **تصميم صارم واحترافي:** واجهة مستخدم مبنية بملف CSS مخصص بالكامل (بدون Bootstrap). تعتمد على 4 ألوان أساسية فقط (أبيض، أسود، رمادي، أصفر) وتخلو تماماً من الرموز التعبيرية (Emojis) لضمان بيئة عمل احترافية، سريعة، وخالية من المشتتات.

## متطلبات التشغيل

* نظام تشغيل Linux (التعليمات أدناه مخصصة لتوزيعة Debian أو Ubuntu)
* Python 3
* بيئة بايثون الوهمية (venv)

## طريقة التشغيل على لينكس ديبيان (Debian Linux)

**1. تحديث النظام وتثبيت المتطلبات الأساسية**
تأكد من تثبيت لغة Python، وأداة تثبيت الحزم `pip`، وأداة إنشاء البيئات الوهمية `venv`:
```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv

cd /path/to/Meswada

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python run.py
